# DataGrapher
A browser-based tool that transforms CSV files and spreadsheet data into interactive network graphs to visually represent patterns and data.

Feature list:

- Drop in a CSV or paste from Google Sheets — no setup needed
- Turns your data into an interactive force-directed network graph
- Size, group, and color nodes by any column
- Click any node to inspect its data in the sidebar
- Export the graph as an SVG
- Light and dark mode
- Runs entirely in the browser — single HTML file, no backend
